# Seesaw
A seesaw example using only HTML, CSS, JS

Overview:

There are 4 text fields. First field show the upcoming weight. Second field and third field show the weight in left and right side. Last field show the angle of the seesaw. Blue area is where the seesaw is located. A clickable area is created where user can drop the weight this area includes above and below the seesaw but exclude the right and left side of the sewsaw. Additionally, the cursor change in this area. Under this blue zone there is a reset button without this button user's actions will be saved in local storage. User can reset weights and and position of the seesaw by clicking reset button. Under the reset button, there is a orange zone which shows the console log. In this area user can observe all the dropped weights along with thier value, location and sides which they belong.  


How to use:

Click the blue area on the seesaw. Observe the interaction. Reset the weights by clicking reset button. Keep track of all the dropped weight in the orange zone.
